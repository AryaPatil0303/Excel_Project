# Excel_Sales_Dashboard

## Project Overview
This project is an interactive Sales Dashboard created using Microsoft Excel.
The dashboard helps analyze sales performance using key metrics, charts, and slicers, making it easy to understand business insights at a glance.

## Tools Used
1. Microsoft Excel
2. Pivot Tables
3. Pivot Charts
4. Slicers & Filters
5. Formulas and Functions

## Key Features
1. Total Sales Analysis  
2. Sales by Region  
3. Sales by Product / Category  
4. Monthly Sales Trend  
5. Interactive dashboard with slicers for easy filtering  

## Dataset
The dataset used in this project contains:
1. Order Date  
2. Product / Category  
3. Region  
4. Sales Amount  
5. Quantity Sold  

## Objectives
1. To practice Excel data analysis and visualization.
2. To build an interactive dashboard.
3. To improve skills in business reporting using Excel.

## Learnings
1. Creating Pivot Tables and Pivot Charts  
2. Designing clean and interactive dashboards  
3. Using slicers for dynamic filtering  
4. Presenting data in a business-friendly way
